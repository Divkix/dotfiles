# dotfile

My configuration across multiple machines

## OS Used: Manjaro

## Run this first:
```
sudo pacman -Syyuu
sudo pacman -S base-devel
```

## Install [yay](https://github.com/Jguer/yay) package helper
```
pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
```
